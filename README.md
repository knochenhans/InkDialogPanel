# InkDialogPanel

Infinity Engine-style dialogs for the Godot engine, utilizing [GodotInk](https://github.com/paulloz/godot-ink) to read Ink files.

## Dependencies

* Godot 4 (mono)
* [GodotInk](https://github.com/paulloz/godot-ink)
